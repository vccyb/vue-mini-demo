1. 如何才能让 obj 变成响应式数据呢？

1. 副作用函数 effect 执行的时候，收集依赖，触发字段 obj.text 的读取操作
1. 当修改 obj.text 的值时候，会触发字段 obj.text 的设置操作，把对应的依赖拿出来执行

关键是拦截对象的读取和设置

1. 将副作用函数存储到一个桶里

```
effect(),读取obj.text => 触发读取操作 => 存储副作用函数到桶里
```

2. 把副作用函数从桶里取出来并执行

```
obj.text = 'xx' => 触发设置操作 => 取出桶里的副作用函数并执行
```

所以关键是啥，拦截对象的读取和设置
=> proxy
